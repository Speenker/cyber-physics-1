# Лабораторная работа 1 CV

## 1. Выбор начальных условий

### 1a. Выбор набора данных и обоснование
Выбран набор данных Building Dataset для задачи многоклассовой классификации изображений бетона по типам дефектов.

### 1b. Выбор метрик качества и обоснование
Для оценки качества используются метрики Accuracy и F1 macro.

- Accuracy показывает долю верных ответов.
- F1 macro усредняет F1 по классам и учитывает каждый класс одинаково, поэтому подходит при возможном дисбалансе классов.

Дополнительно выводятся отчёт по классам и матрица ошибок.

## 2. Создание бейзлайна и оценка качества

### 2a. Обучение моделей из torchvision
Обучены две модели из torchvision:

- Сверточная модель ResNet18.
- Трансформерная модель ViT B 16. Если ViT недоступна, используется Swin T.

Для бейзлайна обучается только классификатор, остальные параметры заморожены.

### 2b. Оценка качества моделей
Оценка выполняется на валидационной выборке по метрикам Accuracy и F1 macro.

Таблица 2b. Результаты бейзлайна

| Модель | Accuracy val | F1 macro val |
|---|---:|---:|
| ResNet18 | 0.835570 | 0.830190 |
| ViT B 16 | 0.890940 | 0.886277 |

## 3. Улучшение бейзлайна

### 3a. Сформулировать гипотезы
Проверены гипотезы улучшения без переобучения, только на этапе инференса:

- TTA на основе TenCrop.
- Ансамбль по логитам ResNet18 и ViT.
- Подбор веса моделей в ансамбле по валидации.

### 3b. Проверить гипотезы
Для каждого варианта выполняется инференс на одной и той же валидации и сравнение по Accuracy и F1 macro.

В качестве улучшенного бейзлайна без переобучения выбран вариант с наилучшим значением F1 macro на val.

| Вариант | Accuracy val | F1 macro val |
|---|---:|---:|

- более сильные аугментации на train,
- label smoothing,
- частичное размораживание слоёв,
- scheduler.

### 3e. Оценить качество моделей с улучшенным бейзлайном
Оценка выполнена на val по тем же метрикам, что и в пункте 2.

### 3f. Сравнить результаты моделей с улучшенным бейзлайном с результатами из пункта 2

|---|---|---:|---:|

Лучший результат в этом сравнении даёт ViT B 16 при улучшенном обучении.
